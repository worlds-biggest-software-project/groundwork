# Project Constitution

## Purpose

This project is a **meta-project**: it manages, researches, and organises a catalogue of candidate software projects. It is NOT a software project in its own right.

## The `projects/` Directory

The subdirectories under `projects/` (e.g. `projects/02-intelligent-test-generator/`) are **candidate project artefacts** — research notes, feature surveys, and READMEs describing software that *could* be built. They are NOT part of this codebase.

**Rules:**

- NEVER edit, create, or delete files inside `projects/` unless explicitly instructed to do so for a specific project.
- NEVER read files inside `projects/` unless explicitly asked to work on that project.
- NEVER offer advice, suggestions, or analysis about the software described in those projects unless specifically asked.
- When running skills (e.g. `/project-create-README`), act only on the specific project named — do not generalise or touch other project directories.
